# Getting Started

## Installation

Install the package using your preferred package manager:

::: code-group

```bash [bun]
bun add @tsports/termenv
```

```bash [npm]
npm install @tsports/termenv
```

```bash [yarn]
yarn add @tsports/termenv
```

:::
